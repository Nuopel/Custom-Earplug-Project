# toolkitsd-acoustmm

`toolkitsd-acoustmm` is the reusable acoustic simulation core extracted from the earplug R&D project.

It provides transfer-matrix building blocks for:

- rigid and viscothermal ducts
- section changes and resonators
- films, membranes, and plate-based lumped elements
- porous and equivalent-fluid layers
- ear-canal and post-processing helpers used in the LOT5 studies

For project-level context, showcase scripts, and theory notes, start from the repository root `README.md` and `LOT5_acoustic_filter_simulation/README_dev.md`.
